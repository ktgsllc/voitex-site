### 第5章 活用パターン集（3〜4ページ）

#### パターンA: コールセンター / カスタマーサポート
- 課題: 入電のうちカスハラ化する割合の把握ができない
- 運用フロー: 全通話取込 → NGワード＋感情検知 → 日次ダッシュボード → 管理者トリアージ
- 成果として期待できる指標（※断定せず例示）:
  - 要注意通話の検出漏れ削減
  - オペレーターの離職兆候の早期把握

#### パターンB: 営業部門（対顧客・対販売店）
- 課題: クレーム化の兆候を営業個人の主観で処理している
- 運用フロー: 訪問/商談録音 → セブンレンズ（営業 + リスク）→ 管理職レビュー
- 「担当替えが遅れる → 被害拡大」を会話相性スコアで先回り

#### パターンC: 窓口対応（自治体・金融・医療）
- 課題: 対面録音の運用と個人情報取扱いの両立
- 運用フロー: 告知徹底 → シングルテナント保管 → 必要時のみ解析 → 監査ログ保持365日

#### パターンD: 経営 / 役員レポーティング
- 月次で「カスハラ関連指標ダッシュボード」を自動生成
- 役員会資料へのコピーペースト運用

> 各パターンに「現場で起きがちな落とし穴」「告知・同意取得の運用上の注意」を併記
